What I learned

During this school year I learned how to use HTML, CSS, and Bootstrap to make websites. Using these I am able to make websites with multiple pages. I can style these web pages with text of different fonts, styles, and colors. I can insert images and buttons into my sites that link tp other pages or sites. Using keyframes I can animate parts of websites. With bootstrap I can add navbars, jumbotrons and use the grid system to make the layout for websites work on devices with different sized screens. I also learned how to use Github to upload my sites.
